    校园应用 ¶

    学习辅助 ¶

    笔记类 ¶

    学会使用 markdown - obsidian 免费的笔记软件，利用主题可以极大美化你的笔记 - Xmind 制作思维导图的软件 - Mkdocs 搭建个人笔记网站，便于整合和上传学习资料 - Hexo 搭建个人博客，主题丰富多元

    文献类 ¶

    学会使用 latex - Overleaf 在线的latex编辑器 - Zotero 文献管理软件
