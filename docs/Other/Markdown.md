# Markdown

markdown 本质是上是对 html 的映射，它可以将 md 元素映射成 html 标签。例如 `## 前言` 实际上转换成 `<h2>前言</h2>`。所以你完全可以用 html 和 css 糊一个自定义 md 样式。

## Admonitions

- note、seealso
- abstract、summary、tldr
- info、todo
- tip、hint、important
- success、check、done
- question、help、faq
- warning、caution、attention
- failure、fail、missing
- danger、error
- bug
- example
- quote、cite
