# Markdown
## 基础语法
### 分点
如果正文中你有两部分要分点，注意符号不能相同
### Table
#### 语法
```markdown
| segment1 | segment2 |
| :-: | :-: |
| content1 | content2 |
```
#### 预览
| segment1 | segment2 |
| :-: | :-:|
| content1 | content2 |

### Highlight
#### 语法
```markdown
<mark>highlight content<mark>
```
#### 预览
<mark>highlight content<mark>

### 收起
#### 语法
```markdown
<details>
<summary>点击展开/收起</summary>
文字内容
</details>
```
#### 预览
<details>
<summary>点击展开/收起</summary>
text content
</details>
