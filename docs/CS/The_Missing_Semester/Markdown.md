# Markdown
## 基础语法
### 分点
如果正文中你有两部分要分点，注意符号不能相同
### 表格
```markdown
| biatou | biaotou |
| :- | :- |
| 表头 |  |
|  |  |
|  |  |

```

### 高亮
#### 语法
```markdown
<mark>文字内容<mark>
```
#### 预览
<mark>文字内容<mark>

### 收起
#### 语法
```markdown
<details>
<summary>点击展开/收起</summary>
文字内容
</details>
```
#### 预览
<details>
<summary>点击展开/收起</summary>
文字内容
</details>
