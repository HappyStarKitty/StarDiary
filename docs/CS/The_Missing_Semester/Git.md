# Git

## Work Flow

工作区 => 暂存区 => 远程仓库

### Setting up

#### git init

```bash
git init <directory>
```

#### git clone

```bash
git clone <repo> <directory>
```

### Saving changes

#### git add

```bash
git add <directory>/<filename>
git add .
```

#### git commit

```bash
git commit -m <commit message>
```

#### git push

```bash
git push origin main
```

### Syncing

#### git push

```bash
git push <remote> <branch>
```

#### git pull

```bash
git pull <remote>
```

### Inspecting

#### git status

```bash
git status
```

#### git log

```bash
git log
```
