## 快捷键

Win+R
  + cmd 命令行
  + sysdm.cpl 系统设置（配置环境变量常用）
  + regedit 注册表
  + control folder 文件夹设置（如显示被隐藏的文件夹）
  + services.msc 服务（sql服务等）
