### 程序填空

题目描述：计算1+2+..n
Hint：这里需要注意一下n为什么是[bp+4]
因为堆栈中除了压入参数n和sum的偏移地址外，call指令还压入了返回地址
堆栈布局如下
+-------------+ <-- 栈顶
|     bp      |
+-------------+
| return addr |
+-------------+
|      n      |
+-------------+
|    *psum    | <-- *psum 即为 offset psum
+-------------+
bp之前是局部变量空间，bp之后是函数参数
