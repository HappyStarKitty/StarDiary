# Chap1 飞机结构设计引论

## 概念内涵

!!! note "飞行器结构"
    飞行器结构：能够**承受和传递载荷**的系统，具有一定的**强度、刚度、质量、寿命和可靠性**

    - 飞机结构：通常包括机身、机翼、平尾、垂尾、短舱吊挂以及相关系统的结构件

    - 火箭结构：火箭各个受力和支承构件的总成

!!! definition "飞行器结构设计"
    飞行器结构设计：指基于结构设计的**原始条件**，按照结构设计的**基本要求**，提出合理的**结构设计方案**，进行具体的**部件和零构件设计、强度与刚度计算、寿命估算和必要的试验**，最后绘制出**结构图纸**，完成相应的**技术文件**，使生产单位能根据这些图纸和文件进行生产

结构优化设计：在给定约束条件下，按某种目标（如重量最轻、成本最低、刚度最大等）求出最好的设计方案

!!! note "飞行器结构、强度、载荷三者关系"
    1. 载荷是结构设计的重要输入，载荷的大小直接影响到结构尺寸甚至结构方案；结构的强度、重量等直接影响载荷大小

    2. 结构和载荷是强度分析的重要输入，结构方案和载荷大小直接影响强度结论

    3. 强度结论对结构和载荷进行反馈，如果强度不满足要求，结构方案需进行优化完善

飞行器结构承受的载荷可归类为五种形式：拉伸、压缩、弯曲、剪切、扭转

飞机结构设计的重点，就是要**根据载荷形式，设计合理的结构形式**，高效率承受飞机载荷

典型的结构形式有：梁、肋、蒙皮、长桁

外载荷：机翼结构、尾翼结构、机身结构、起落架结构

## 结构设计

研制过程：论证阶段、方案阶段、工程研制阶段、设计定型阶段、生产定型阶段

飞机结构设计原始条件：结构外载荷和受力特点、结构外型和空间协调、结构的使用条件、结构的生产条件

基本要求：气动要求、重量要求、使用维护要求、工艺要求、经济性要求、可靠性要求

重量要求是主要要求，飞机结构重量占全机重量之比是衡量飞机结构设计先进性的最重要指标
