# Chap4 翼面结构设计与分析

## 机翼的功用和设计要求

!!! note "机翼的功用"
    1. 产生升力，保证飞机战术技术所规定的飞行和机动性能要求
    2. 保证飞机横向稳定性、操纵性以及增升、增阻效能
    3. 安装起落架、发动机、武器等
    4. 装载燃油、军械、设备等

!!! note "机翼的构成"

    1-骨架翼梁；2-腹板和桁条；3-襟翼；4-扰流片；5-副翼；6-蒙皮；7-前缘缝翼；8-挂架；9-翼肋

气动、强度、刚度、寿命、重量要求是机翼的主要要求，这是由机翼的功用、载荷和外形特征因素所决定的


## 机翼的载荷与内力

!!! note "机翼的外载荷"
    - 空气动力
        + 分布载荷，以吸力和压力的形式直接作用在翼面的蒙皮上，形成翼面的升力和阻力
        + 升力是翼面的最主要外载荷
    - 翼面结构质量力
        + 翼面结构本身的质量力（包括重力和惯性力）作用在翼面整个容积上
        + 可近似认为与翼面的弦长成正比
    - 其它部件和外挂传入的集中载荷
        + 通过各自连接接头以集中载荷的形式传给翼面
        + 整体油箱的燃油载荷为分布载荷  

     上述三类载荷按分布方式可分为

     * 分布载荷，包括气动载荷、翼面结构质量力、燃油载荷，按一定规律分布作用在翼面结构上，是翼面的主要载荷形式

     * 集中载荷，由其他部件、外挂物的接头传入，大小与过载系数有关，方向大都与升力相反，对翼面结构起卸载作用



## 机翼主要受力构件及其结构

!!! note "蒙皮"
    - 形成流线形的机翼外表面，并参与机翼的总体受力

    - 承受局部气动载荷
    - 参与总体扭矩的传递
    - 参与总体弯矩的传递

!!! note "翼梁"
    - 翼梁一般有缘条、腹板和支柱构成，承受剪力和弯矩
    - 对于梁式机翼，是最重要的纵向元件
    - 承受总体剪力——腹板剪流
    - 参与总体弯矩的传递——上下缘条

!!! note "长桁"
    - 支持和加强蒙皮，承受局部空气力载荷
    - 将翼肋互相联系起来
    - 承受由弯曲产生的正应力

!!! note "纵墙/腹板"
    - 封闭作用
    - 参与总体扭矩的传递
    - 参与总体剪力的传递

!!! note "翼肋"
    - 维持机翼形状
    - 支持长桁和蒙皮承受气动载荷，提高其稳定性
    - 传递气动力至梁腹板、剪流至蒙皮
    - 承受、传递集中载荷
    - 不参与总体力的传递

!!! note "翼肋上的部分特征"
    1. 孔-减轻重量，通过拉杆、电缆等
    2. 孔边带有弯边，为了提高翼肋稳定性和刚度
    3. 有些腹板上压有凹槽，作用相当于弱支柱，增加腹板稳定性和侧向刚度
    4. 分段，为了便于同翼梁腹板或者墙腹板连接
