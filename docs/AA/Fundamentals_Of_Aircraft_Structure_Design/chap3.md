# Chap3 飞机的外载荷和设计情况

## 飞机的外载荷和过载

### 外载荷

!!! note "主要外载荷"
    飞机外载荷：飞机在使用过程中作用于飞机上的各种力的统称

    - 空气动力
    - 质量m有关的质量力
    - 起飞和着陆受到的空气动力和冲击载荷
    - 推力系统产生的载荷
    - 水平阵风和垂直阵风产生的载荷
    - 运载装置或工具作用于飞行器上的载荷
    - 大气层高速飞行产生的气动加热温度载荷与在太空中飞行时不均匀辐射引起的温度载荷
    - 外来物的撞击载荷
    - 失事时的非正常载荷

!!! note "载荷的划分"
    - 按作用状态分为静载荷、动载荷
    - 按分布形式分为集中载荷、分布载荷
    - 按载荷性质分为质量力、质量无关力

### 过载


!!! definition "过载"
    过载：飞机所受除重力之外的外力总和与飞机重力之比（所有表面力合力与飞机重量$G$之比），用$n$表示

    $$n=R_{bi}/G$$

    过载$n$为矢量，方向与表面力合力$R_{bi}$一致

!!! note "不位于质心处的质量过载"
    飞行器绕质心旋转所引起的加速度为$\Delta a_x$和$\Delta a_y$，相应产生的附加过载$\Delta n_x$和$\Delta n_y$

    附加过载与质量到质心距离$x$成正比

!!! note "大气紊流引起颠簸过载"
    又称突风过载或阵风过载

!!! note "过载系数"
    与过载一致，但是是比值

## 安全系数与设计载荷

使用载荷：正常使用中所允许到达的最大载荷，或称限制载荷

设计载荷：设计时结构需要承受高于使用载荷的载荷，即对静强度问题采用设计载荷法或破坏载荷法，飞行器在该载荷下不应破坏，又称极限载荷

!!! note "破坏载荷法"
    强度条件为

    $$F_d \le F_e \cdot f$$

    或

    $$\sigma_d \le \sigma_b$$

    $F_d$——设计载荷；$F_e$——使用载荷；$f$——安全系数；$\sigma_d$——设计应力；$\sigma_b$——材料破坏应力

!!! definition "安全系数"
    安全系数：设计载荷与使用载荷之比

    $$f=\frac{F_d}{F_e}=\frac{F_d}{nG}$$
