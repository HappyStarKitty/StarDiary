# Chap5 机身结构分析与设计

## 机体结构的功用、结构特点和设计要求

机体是受力基础，外载荷主要是各部件传入的集中力

机体上大开口和舱门比较多，必须布置适当构件来将集中力扩散，并以分布力形式传给其他元件

## 机体结构的载荷及其平衡

主要外载荷：各部件传入的集中载荷，质量力，空气动力，增压载荷，振动载荷，冲击载荷

## 典型结构形式及传力分析

结构形式：一般采用刚性薄壁空间或空间桁架结构，主要有桁架式结构、桁条式结构（单块式结构）、桁梁式结构、梁式结构、硬壳式结构、智能结构

!!! note "机身典型结构布局"
    以Su27为例

    - 前机身
        + 由设备舱、驾驶舱和前起落架舱组成
        + 铝合金半硬壳式结构

## 加强框的受力分析和设计

加强框形式：环形刚框式加强框、腹板式加强框、构架式加强框
