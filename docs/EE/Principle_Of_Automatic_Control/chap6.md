# Chap6 线性系统的校正方法

## 系统的设计与校正问题

### 校正方式

控制系统校正方式可分为串联校正、反馈校正、前馈校正和复合校正四种

### 基本控制规律

在工业过程控制系统中，广泛使用PID控制器。PID控制器各部分参数的选择，应使I部分发生在系统频率特性的低频段，以提高系统的稳态性能；而使D部分发生在系统频率特性的中频段，以改善系统的动态性能
