# Chap1 绪论

- 信号$x(t)$是时间$t$的函数（实值或复值）
    - 若$t \in R$，称为连续信号，记为$x(t)$
    - 若$t \in Z$，称为离散信号，记为$x(k)$

- 系统是从一个信号（称为输入）到另一个信号（称为输出）的映射

- 周期信号
    - 对于连续信号，若存在$T >0$，使$X(t)=x(t+nt)$ n为整数
    - 对于离散信号，若存在$T>0$，若存在正整数$N$，使$X(n)=x(n+kN)$ k为整数
- 非周期信号
    - 不具有周期的信号，可视为周期无穷大

信号的能量

$$W=\lim_{T \rightarrow\infty}\int_{-T}^{T}|x(t)|^2 dt$$

信号的功率

$$P=\lim_{T \rightarrow\infty}\frac{1}{2T}\int_{-T}^T x^2(t) dt$$

能量信号：W为有限值，P=0

功率信号：P为不等于0的有限值
