# Chap5 集成运算放大器

## 集成运放的基本组成

### 概述

!!! note "集成运放"
    集成运放基本组成主要有四个部分：输入级、中间级、输出级、偏置电路

    - 输入级：组成集成运放的关键部分，通常要求有高输入电阻、低漂移和高抗干扰能力，常采用差分放大电路
    - 中间级：亦称为电压放大极，主要使集成运放获得很高的电压放大倍数，常由一级或多级电压放大电路组成
    - 输出级：主要使集成运放有较强的带负载能力，通常采用互补对称式电路
    - 偏置电路：为各级电路提供偏置电流
